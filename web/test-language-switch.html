🌐 DeerFlow 语言切换测试

测试步骤

  1. 打开 http://localhost:3001
  2. 查看页面是否显示英文内容
  3. 点击右上角的语言切换器（地球图标）
  4. 选择 "🇨🇳 中文"
  5. 验证页面内容是否切换为中文
  6. 刷新页面，验证语言选择是否保持

预期的中文翻译内容

  • 主标题："深度研究触手可及"
  • 副标题："认识 DeerFlow，您的个人深度研究助手..."
  • 按钮："开始使用"、"了解更多"
  • 功能标题："核心功能"
  • 案例研究："案例研究"

聊天页面测试

  1. 访问 http://localhost:3001/chat
  2. 切换到中文
  3. 验证欢迎消息："👋 您好！"
  4. 验证描述文本包含"欢迎使用 🦌 DeerFlow"

✅ 成功标准

  • 页面加载无错误
  • 语言切换器可见且可点击
  • 切换到中文后，所有文本都显示为中文
  • 刷新页面后语言选择保持不变
  • 在不同页面间导航时语言保持一致
